This is a fully functional two-sided marketplace simulation with 3 independent advertisers and 3 independent publishers. You're currently logged in as Nike. Each advertiser and publisher has their own unique campaigns, bids, and analytics.

🏃 Nike 🍔 McDonald's ₿ Coinbase

📱 TechDaily 💪 FitZone ₿ CryptoNews

🎯 Campaign Targeting is Specific & Isolated

🍔 McDonald's Example

If McDonald's creates a campaign targeting Fitness Publishers, that campaign will ONLY appear in:

✓ FitZone's dashboard ✗ TechDaily's dashboard ✗ CryptoNews's dashboard ✗ Nike's advertiser dashboard
🏃 Nike Example

If Nike targets Tech + Crypto Publishers, that campaign appears in:

✓ TechDaily's dashboard ✓ CryptoNews's dashboard ✗ FitZone's dashboard ✗ Other advertisers' dashboards
Key Point: Each advertiser sees ONLY their own campaigns. Each publisher sees ONLY campaigns targeting their category.
